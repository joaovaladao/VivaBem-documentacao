    Épico 06 - Área mais opções

    1. US25 - Alterar as configurações

    Eu, como usuário, quero alterar as configurações para que eu organize e fique ao meu gosto.


    Critérios de aceitação:

    • Deve ter um botão configurações para configurar o aplicativo.
    • Configurar os sons dos alarmes e notificações.
    • Gerenciar senha

    2. US26 - Cadastrar um médico para contato

    Eu, como usuário, quero cadastrar um médico para contato para que eu possa ligar em caso de emergência.


    Critérios de aceitação:

    • Função para cadastrar um contato médico para emergências

    3. US27 - Visualizar as novidades

    Eu, como usuário, quero visualizar as novidades para que eu possa me manter atualizado com as notícias do mundo da saúde.


    Critérios de aceitação:

    • Deve ter conexão com a internet
    • Poder receber notificações de novas notícias

    4. US28 - Rendimentos

    Eu, como usuário, quero ver o rendimento do mês para que eu veja se estou cumprindo com as metas.


    Critérios de aceitação:

    • Ter cadastrado um medicamento para ver o rendimento de uso
    • Poder visualizar o rendimento dos meses passados e comparar com o atual
    • Responder o questionário diário

    5. US29 - Visualizar os termos de uso

    Eu, como usuário, quero visualizar os termos de uso para que eu concorde com os termos dados.


    Critérios de aceitação:

    • Poder visualizar os termos de uso
