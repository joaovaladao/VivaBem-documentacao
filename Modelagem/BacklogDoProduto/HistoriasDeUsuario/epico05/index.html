    Épico 05 - Área de Exames

    1. US21 - Realizar o cadastro de um exame

    Eu, como usuário, quero realizar o cadastro de um exame para que eu tenha o exame registrado no aplicativo.

    Critérios de aceitação:

    • Deve ter um botão "adicionar novo exame" para cadastrar o exame novo
    • Deve ter um campo para inserir o tipo do exame
    • Deve ter um campo para inserir a data do exame
    • Deve ter um campo para inserir o local do exame
    • Deve ter um campo para inserir a Unidade de saúde solicitante do exame
    • Deve ter um campo para inserir o nome do exame

    2. US22 - Ser notificado com antecedência o horário do exame

    Eu, como usuário, quero ser notificado com antecedência o horário do exame para que eu não esqueça e não perca o exame.

    Critérios de aceitação:

    • Deve ter uma notificação avisando que se aproxima do horário do exame
    • Deve ter um campo informando o nome do exame
    • Deve ter um campo informando o local do exame
    • Deve ter um campo informando o data e horário do exame

    3. US23 - Consultar a lista de exames

    Eu, como usuário, quero consultar a lista de exames para que eu veja todos os exames que foram realizados.

    Critérios de aceitação:

    • Deve ter uma lista de todos os exames que o usuário realizou e agendou
    • Deve ter uma opção selecionar um exame da lista para ver suas informações
    • Deve ter uma opção de alterar as informações de um exame da lista
    • Deve ter uma opção de apagar um exame da lista

    4. US24 - Confirmar realização de exame, consulta ou compromissos

    Eu, como usuário, quero confirmar realização de exame, consulta ou compromissos para que eu controle as ações, dependendo da situação.

    Critérios de aceitação:

    • Deve ter um alerta no horário marcado do exame
    • Deve ter um botão "confirmar" caso seja feito o exame
    • Deve ter um botão "negar" caso não seja feito o exame
